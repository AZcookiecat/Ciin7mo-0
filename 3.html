mysql报错之 ERROR 1045（2800）

作者：Ciin7mo 发布于：2025年3月21日

问题描述：ERROR 1045 (28000): Access denied for user 'root'@'localhost' (using password: YES) ERROR 1045 (28000): Access denied for user 'root'@'localhost' (using password: NO)

解决方案：

1.用管理员身份运行命令行 输入 net stop mysql 关闭mysql服务

2.在my.ini中添加skip-grant-tables 用于跳过密码登录

3.用管理员身份运行命令行 输入 net start mysql 开启mysql服务

4.输入mysql -u root -P 3306 -h localhost -p 回车输入密码 会直接登陆成功

5.修改密码
先输入use mysql;
然后输入update user set authentication_string=password('你的密码') where user='root';
最后flush privileges; 刷新权限

6.删除skip-grant-tables 注释
